c. Contenido

Estilos internos y Externos

Los estilos aplicados de manera interna son aquellos que se generan desde el HTML, especificamente en las etiquetas base como Head y body, bajo el contexto de la etiqueta "style".

Estilos internos y Externos

Son estilos que se pueden aplicar desde un archivo alterno, el cual, se puede referenciar a etiquetas, clases y de algunas otras formas para mantener el código mejor organizado, planteando estructuras más especificas a través de selectores, aplicando caracteristicas y sin mezclar estilos, ya que con el estilo Interno suele pasar ese tipo de problemas, en este estilo solamente se entura desde el archivo HTML con un "link rel" a la hoja de estilos creada previamente.

Selectores CSS

Los selectores son puntos de referencia para establecer conexión directa con archivo HTML y con una herramienta especifica, algunos de ellos son como menciona (MDN web docs , 2019):

  • Selector de tipo Selecciona todos los elementos que coinciden con el nombre del elemento especificado.
  • Selector de clase Selecciona todos los elementos que tienen el atributo de class especificado.
  • Selector de ID Selecciona un elemento basándose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento.
  • Selector universal Selecciona todos los elementos. Opcionalmente, puede estar restringido a un espacio de nombre específico o a todos los espacios de nombres.
